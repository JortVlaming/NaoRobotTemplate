Language managementÂ¶

Note: you may not see this screen.

If the selected language at the beginning of the configuration is not installed on the robot, you may face an error with the language update:

To pass the error: set back the robot in English by clicking the button on the left, “Configure the language in English”.

As a consequence, Robot Settings will switch back in English.

If you want NAO talks in another language, contact the Customer support.

For further details, see: Contacting customer support for NAO.

End of NAO configuration on Robot SettingsÂ¶

When the screen below appears, the initial configuration is completed:

  • Autonomous life is installed,
  • English language is installed.

To complete the configuration:

Step Action

Click the Start! button.

The Robot Settings now displays the settings menu:

If you had previously set up a WiFi connection

You can now remove the Ethernet cable and put back the hatch behind the head of the robot.

As a consequence, Robot Settings will lost the connection and the robot will switch to the previously registered WiFi network.

Connect your computer on this WiFi network and ask again NAO his IP Address in order to reconnect Robot Settings to the robot.

If you haven’t set up a WiFi connection previously

When you want to connect to NAO again, plug the Ethernet cable and ask for NAO IP address.

Once connected in Robot Settings, you can setup a WiFi connection in the NAO settings.
